# 🐷 Pig Escape (เกมหมูหนี) - OOP Final Project

## 📌 Overview
**Pig Escape (หมูหนี)** เป็นเกมแนวแก้ปริศนาสไลด์บล็อก (Sliding Block Puzzle) สำหรับผู้เล่น 1 คน (Single-player) ผู้เล่นจะต้องใช้ไหวพริบในการเลื่อนขยับสิ่งกีดขวางต่างๆ บนกระดาน เพื่อเคลียร์เส้นทางและพาน้องหมูหลบหนีออกทางประตูทางออกให้สำเร็จ [cite: 59]

* **Platform:** PC (Windows / macOS / Linux) รันเป็น Desktop Application [cite: 6]
* **Tech Stack & Libraries:** พัฒนาด้วยภาษา **C++** และใช้ Library **Raylib** ในการเรนเดอร์กราฟิกแบบ 2D และจัดการ UI/Input [cite: 6]

---

## 📖 Rules (กติกาของเกม)
ผู้เล่นจะต้องพา "น้องหมู (P)" ไปยังประตูทางออก (Exit) ที่อยู่ฝั่งขวาของกระดาน [cite: 67, 76] โดยมีเงื่อนไขดังนี้:
1. **การเคลื่อนที่:** สิ่งกีดขวางและหมูสามารถขยับได้ตามแนวแกนของตัวเองเท่านั้น (แนวนอน หรือ แนวตั้ง) [cite: 70]
2. **ข้อห้าม:** ไม่สามารถหมุนสิ่งกีดขวางได้ และห้ามขยับทะลุหรือทับซ้อนกับบล็อกชิ้นอื่น [cite: 73, 74]
3. **เงื่อนไขแพ้ (Game Over):** * ทำการสไลด์เกิน "โควตาจำนวนก้าวเดิน" ที่ด่านนั้นๆ กำหนดไว้ 
   * ไม่สามารถพาน้องหมูหนีได้ทันก่อนที่ "เวลา" จะนับถอยหลังจนหมด

## 🎮 Game Manual & How to Play (คู่มือการเล่น)
1. [cite_start]**เริ่มเกม:** เมื่อเปิดโปรแกรม ให้กรอกชื่อผู้เล่นในหน้า Login ด้วยคีย์บอร์ด และคลิกเมาส์ที่ปุ่ม "Start Game" [cite: 14]
2. **วิธีควบคุม (Input):**
   * [cite_start]ใช้ **"เมาส์คลิกซ้าย (Left-Click)"** ไปที่บล็อกที่ต้องการขยับ [cite: 14]
   * หากคลิกที่สิ่งกีดขวาง (บล็อกสี่เหลี่ยมสีต่างๆ) บล็อกนั้นจะทำการ **"สไลด์จนสุดขอบกระดาน"** หรือจนกว่าจะชนกับบล็อกชิ้นอื่น ในทิศทางของแกนนั้นๆ อัตโนมัติ
   * [cite_start]หากคลิกที่น้องหมู (บล็อกสีชมพู) น้องหมูจะขยับไปข้างหน้าทีละ 1 ก้าว เพื่อเข้าสู่ประตูทางออก [cite: 14]
3. **การประมวลผล:** ทุกครั้งที่มีบล็อกขยับสำเร็จ ตัวแปร `Moves` จะเพิ่มขึ้น 1 ก้าว ผู้เล่นต้องสังเกตเวลา (Time) ที่นับถอยหลังอยู่ด้านบนหน้าจอเสมอ

## 💻 Software Architecture & OOP Concepts
[cite_start]โปรเจกต์นี้ได้รับการออกแบบตามหลักการ Object-Oriented Programming (OOP)  ดังนี้:
* **Encapsulation:** มีการซ่อนสถานะของเกม (เช่น `currentMove`, `timeRemaining`) ไว้เป็น `private` ในคลาส `GameManager` และ `Level` โดยควบคุมผ่าน public methods[cite: 125].
* **Inheritance:** คลาส `Pig` (หมู) และ `Obstruction` (สิ่งกีดขวาง) มีการสืบทอดคุณสมบัติพิกัดและขนาดมาจาก Base Class นามธรรมที่ชื่อว่า `GamePiece` (สืบทอดอย่างน้อย 2 จุด)[cite: 26, 124].
* **Polymorphism / Collections:** คลาส `Board` ทำการเก็บชิ้นส่วนทุกประเภทไว้ใน Collection เดียวกันผ่าน `vector<GamePiece*> pieces` ทำให้ง่ายต่อการสั่งรันลอจิกการเช็กการชน (Collision) และประมวลผลการสไลด์บล็อก
